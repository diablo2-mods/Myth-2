Mercenaries
 
 
*Can be fully equipped.
*(Normal, Torment 1, Inferno 1) Life, Defense, Attack Rating, Damage and Elemental Resist +0%
*(Nightmare, Torment 2, Inferno 2) Life, Defense, Attack Rating, Damage and Elemental Resist +10%
*(Hell, Torment 3, Inferno 3) Life, Defense, Attack Rating, Damage and Elemental Resist +20%
 
ACT I - Rogue Scouts (Bow or Crossbow)
- Fire: Innersight + Slow Missiles + Fire Arrow + Exploding Arrow + Immolation Arrow.
- Cold: Innersight + Slow Missiles + Cold Arrow + Ice Arrow + Freezing Arrow.
- Lightning: Innersight + Slow Missiles + Lightning Arrow + Lightning Bolt + Lightning Fury.
- Combat: Innersight + Slow Missiles + Magic Arrow + Poison Bolts + Plague Missile
 
ACT II - Desert Mercenaries (Spear or Polearm)
- Combat: Jab + Vengeance + Lightning Strike + Shock Wave + Iron Maiden.
- Defensive: Jab + Vengeance + Cold Strike + Shock Wave + Life Tap.
- Offensive: Jab + Vengeance + Fire Strike + Shock Wave + Amplify Damage.
 
ACT III - Eastern Sorcerors (Sword and Shield)
- Fire: Fire Bolt + Fire Ball + Hydra + Molten Boulder + Meteor.
- Cold: Ice Blast + Glacial Spike + Frost Nova + Frozen Orb + Blizzard.
- Lightning: Charged Bolt + Lightning + Nova + Chain Lightning + Lightning Breath.
- Combat: Teeth + Bone Spear + Bone Spirit + Tornado + Magic Rain.
 
ACT V - Barbarians (Sword)
- Combat: Bash + Stun + Zeal + Berserk + Battle Cry
- Defensive: Bash + Stun + Zeal + Vengeance + Howl
- Offensive: Bash + Stun + Zeal + Sacrifice + War Cry
