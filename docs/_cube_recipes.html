Others
 
Quest Items:
Staff of Kings + Viper amulet = Horadric Staff
Khalim Flail + Khalim Heart + Khalim Eye + Khalim Brain = Super Khalim Flail
Wirt's Leg + Tome of Town Portal = Secret Cow Level
 
Rejuvenation Potions:
3 Any Health Potions + 3 Any Mana Potions + 1 Chipped Gem = 1 Small Rejuvenation Potion
3 Any Health Potions + 3 Any Mana Potions + 1 Standard Gem = 1 Full Rejuvenation Potion
3 Small Rejuvenation Potion = 1 Full Rejuvenation Potion
 
Gem Upgrade:
Same gem x3 = next higher gem quality
 
Rune Upgrade / Downgrade / Change:
Same rune x2 = next higher rune
Rune + Key = next lower rune
Rune + Perfect Gem = The other side of the same rune
 
Quality of Items:
Low Weapon / Armor + 1 Eld Rune + 1 Chiped Gem = Normal Weapon / Armor
Normal Weapon / Armor + 3 Any Perfect Gems = Magic Weapon / Armor
Magic Weapon / Armor + 1 Amn Rune + 1 Perfect Topaz = Low Level Rare Weapon / Armor
Magic Weapon / Armor + 1 Hel Rune + 1 Perfect Topaz = High Level Rare Weapon / Armor
Base ( Normal or Superior ) Weapon / Armor + 1 Ral Rune + 1 Shael Rune + 1 Perfect Sapphire = Exceptional Version of Weapon / Armor +5 Required Level
Base ( Magic, Rare or Crafted ) Weapon / Armor + 1 Ral Rune + 1 Shael Rune + 1 Perfect Amethyst = Exceptional Version of Weapon / Armor +5 Required Level
Base ( Unique or Set ) Weapon / Armor + 1 Ral Rune + 1 Shael Rune + 1 Perfect Diamond = Exceptional Version of Weapon / Armor +5 Required Level

Exeptional ( Unique or Set ) Weapon / Armor + 1 Ko Rune + 1 Mal Rune + 1 Perfect Diamond = Elite Version of Weapon / Armor +7 Required Level
     
Socketed Weapon / Armor:
Normal or Superior Weapon / Armor ( without sockets ) + 1 Ral Rune + 1 Amn Rune + 1 Perfect Bloodstone = Socketed Normal or Superior Weapon / Armor: 1-6 sockets
Magic Weapon / Armor ( without sockets ) + 1 Ral Rune + 1 Amn Rune + 1 Perfect Bloodstone = Socketed Magic Weapon / Armor: 1-4 sockets
Rare Weapon / Armor ( without sockets ) + 1 Ral Rune + 1 Amn Rune + 1 Perfect Bloodstone = Socketed Rare Weapon / Armor: 1-3 sockets
Crafted Weapon / Armor ( without sockets ) + 1 Ral Rune + 1 Amn Rune + 1 Perfect Bloodstone = Socketed Crafted Weapon / Armor: 1-2 sockets
Set or Unique Weapon / Armor ( without sockets ) + 1 Ral Rune + 1 Amn Rune + 1 Perfect Bloodstone = Socketed Set or Unique Weapon / Armor: 1 socket
     
Repair, Recharge, and Removal of Items:
An Item + 1 Ort Rune + 1 Flawed Gem = Fully Repaired and Recharged Item
Throwing Weapon or Quivers + 1 Amn Rune + 1 Flawless Gem = Replenishes Quantity
Any Socketed Item + 1 Hel Rune + Town Portal Scroll = Remove items from sockets
 
Affix Rerolling:
Magic Item + 1 Amn Rune + 1 Perfect Skull = Magic Item with new affixes
Rare Item + 1 Um Rune + 3 Perfect Skulls = Rare Item with new affixes
